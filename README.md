Gesture Control Prototype (OOP Demo)

This is a Python-based final project demonstrating the 4 pillars of Object-Oriented Programming (OOP)—Abstraction, Encapsulation, Inheritance, and Polymorphism—applied to a computer vision and gesture-tracking application structure.

## Project Reference
As required, this conceptual OOP framework is based on existing Python hand-tracking implementations utilizing OpenCV and Google MediaPipe. 

- **Source / Reference Link:** https://github.com/google/mediapipe
